# Bevy Jam 7

## Theme is "Extremely Incohesive Fever Dream"

### Requirements
- `rust`

### Requirements (for wasm builds)
- `rustup target install wasm32-unknown-unknown`
- `cargo install wasm-server-runner`

### Building and Running
- `cargo run --target wasm32-unknown-unknown` (for wasm builds)
- `cargo run` (for native builds)
